# RediPoll

Angular, ExpressJS ve Redis kullanarak geliştirdiğim anket uygulamasıdır. Kullanıcılar anket oluşturabilir, anketlere oy verebilir ve anket sonuçlarını görüntüleyebilirler.

## Kullanılan Teknolojiler
- Angular - TypeScript
- ExpressJS - NodeJS
- Redis
